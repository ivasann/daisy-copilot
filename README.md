## Getting Started

1. Clone this repository
2. (Optional backend) install and run the API:
   - `pip install -r requirements.txt`
   - `cd backend`
   - `uvicorn main:app --reload`
3. Open `website/web.html` in your browser
4. If the API is running, the app uses it. If not, it falls back to LocalStorage.
